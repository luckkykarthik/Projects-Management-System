Project Management System Guidelines

Getting Started

  • Sign in to your account or create a new account if you don't have one.
  • Create a new project by clicking on the "New Project" button.
  • Fill in the project details such as name, description, start date, and end date.
  • Click "Save" to create the project.

Managing Projects

Create Tasks

  • Select a project from the project list.
  • Click on the "Add Task" button.
  • Enter the task details such as title, description, due date, and assignee.
  • Click "Save" to create the task.

Edit Tasks

  • Click on a task from the task list.
  • Click on the "Edit" button.
  • Modify the task details as needed.
  • Click "Save" to update the task.

Track Progress

  • View the progress bar on each task to track its completion status.
  • Mark a task as complete by checking the checkbox next to it.

Collaboration

Invite Team Members

  • Navigate to the project settings page.
  • Enter the email addresses of the team members you want to invite.
  • Click "Invite" to send them an invitation.

Communication

  • Use the project's discussion board to communicate with team members.
  • Create new discussion threads or reply to existing threads.

Reporting

Generate Reports

  • Go to the reporting section.
  • Select the project and date range for the report.
  • Click "Generate Report" to generate the project report.

Export Reports

  • Once the report is generated, click on the "Export" button.
  • Select the desired format (e.g., PDF, Excel) to export the report.
